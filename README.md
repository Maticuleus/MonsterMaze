# Monster Maze
Navigate through the maze and fight monsters.

## Controls
**Z** - light shield (deflects light projectiles) <br />
**X** - dark shield (deflects dark projectiles) <br />
**SPACE BAR** - combo attack <br />
**MOUSE DOWN** - movement
